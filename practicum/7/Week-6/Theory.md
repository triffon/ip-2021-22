## Как да вмъкваме елемент в масив:

След като сме се посигурили, че масивът е с достатъчно голям капацитет да приеме нов елемент:

```c++
for (int i = arr_len; i > pos; i--) { // отместваме елементите                                      
                                       // с 1 надясно
		arr[i] = arr[i - 1];
}

arr_len++;
arr[pos] = elem;
```
